Mac users:
1. Install pip (https://pip.pypa.io/en/stable/installation/)
2. Open new terminal for project folder
3. source venv/bin/activate (enter in terminal)
4. cd ClearviewCRM (enter in terminal)
5. python manage.py runserver (enter in terminal)
6. Enter http://127.0.0.1:8000/ in web browser
7. Enter CTRL+C to quit

Windows users:
1. Install pip (https://pip.pypa.io/en/stable/installation/)
2. Open new terminal for project folder
3. Create virtual environment by entering 'python -m venv myvenv' in terminal
4. myvenv\Scripts\activate (enter in terminal)
5. pip install django (enter in terminal)
6. pip install django-crispy-forms (enter in terminal)
7. pip install crispy-bootstrap4 (enter in terminal)
7. cd ClearviewCRM (enter in terminal)
8. python manage.py runserver (enter in terminal)
9. Enter http://127.0.0.1:8000/ in web browser
10. Enter CTRL+C to quit
